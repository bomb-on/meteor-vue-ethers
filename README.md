# meteor-vue-ethers

```shell
# bootstrap
$ meteor create --vue meteor-vue-ethers --release 3.0.1

# add accounts-base
$ meteor add accounts-base

# att ethers and related
$ meteor npm install ethers @metamask/eth-sig-util

# update meteor-vite version
$ meteor npm uninstall meteor-vite && meteor npm install meteor-vite
```
